Display

O display nos ajuda a modificar os elementos na tela. Alguns elementos oculpam uma linha inteira independêntemente do seu tamanho.

Exemplo: uma li ocupa uma linha inteira, mesmo se ela tiver somente um caracter.

Display Block

O display: block faz um elemento que ocupa uma linha inteira, ocupar somente um espaço determinado nós devemos definir esse espaço no CSS, ficando o restante da linha vázio.

Display Inline

O display: inline faz com que elementos que ocupam uma linha inteira ficarem na mesma linha. Não aceitam a mudança de width e height. Os elementos definidos como display: inline sempre ocuparam somente o seu espaço, não deixando espaços vázios.

Display Inline-Block

O display: inline-block é usado para deixarmos os elementos em linha e ao mesmo tempo, definirmos valores para width e height.

Display None

Já o display: none é usado para ocultarmos os elementos. O usuário não verá o elemento na tela, mas ele existe no HTML.
